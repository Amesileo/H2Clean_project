# H2Clean Website

Modern, high-converting website for H2Clean (h2clean.co.uk)

## Features
- Phone-first CTA strategy ("Call for a Free Quote")
- GSAP scroll animations
- Tailwind CSS
- Lucide-ready (add via npm if using Vite)
- Supabase-ready dynamic gallery (instructions below)

## Supabase Setup (Client Photo Uploads)
1. Create free project at supabase.com
2. Create `gallery_images` table with columns: id, url, caption, created_at
3. Enable Storage bucket named `gallery`
4. Use the Supabase JS client to fetch images in the gallery section.

## Quick Start
Open `index.html` in browser. 

For full TypeScript/Vite version, expand the `src/` folder.
